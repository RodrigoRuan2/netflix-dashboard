# Netflix Dashboard

Um dashboard estilo Netflix construído com React + Vite e a API do TMDB.

## Funcionalidades

- 🎬 Hero rotativo com filmes em alta
- 📋 Múltiplas filas de filmes e séries (populares, top rated, em cartaz)
- 🔍 Busca em tempo real de filmes e séries
- 🎭 Explorar por categoria e gênero
- 📄 Página de detalhes com elenco, trailer e similares
- 💀 Skeletons de carregamento
- 📱 Responsivo para mobile

## Setup

```bash
npm install
npm run dev
```

## Variáveis de ambiente

Crie um arquivo `.env` na raiz:

```
VITE_TMDB_KEY=sua_chave_aqui
```

Obtenha sua chave em: https://www.themoviedb.org/settings/api
